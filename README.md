# Universal Orbit Components
Proof of concept for universal components.
Based on [react-native-web](https://github.com/necolas/react-native-web).

# Start
## Desktop
```bash
yarn storybook
```

## Mobile

After running `yarn`, you also need to link assets using:

```bash
react-native link
```

### iOS

```bash
yarn ios
```

### Android
```bash
yarn android
```
